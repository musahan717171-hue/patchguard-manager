PATCHGUARD MANAGER - AUTOMATIC LAUNCH

Use this file for automatic launch:

START_PATCHGUARD.bat

What it does:
1. Creates venv if needed
2. Installs dependencies if needed
3. Runs makemigrations
4. Runs migrate
5. Starts Django server on 0.0.0.0:8000
6. Opens http://127.0.0.1:8000 automatically

Local computer:
http://127.0.0.1:8000

Other device in same Wi-Fi:
http://YOUR_LOCAL_IP:8000

Demo login:
Username: admin
Password: admin12345

Important:
Keep the BAT window open while using the application.
If Windows Firewall asks permission, click Allow Access.
